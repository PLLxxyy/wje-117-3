# 马拉松训练计划工具

全栈马拉松训练计划工具，根据目标赛事和个人水平自动生成分阶段训练计划，支持每日打卡和教练点评。

## 功能简介

- **生成训练计划**：填写目标赛事、完赛目标成绩、当前水平和每周可训练天数，系统自动生成分阶段训练计划（轻松跑/间歇跑/长距离跑/节奏跑/休息日）
- **训练日历**：以月历形式展示每天的训练计划和完成状态，完成标绿、跳过标灰，支持按周导航
- **每日打卡**：记录实际距离、用时和身体感受，系统根据完成情况调整后续强度
- **跑量统计**：周跑量柱状图（纯 CSS 实现），展示每周跑量趋势
- **教练端**：查看学员训练情况和统计数据，给学员留言和调整建议
- **管理后台**：用户管理、训练模板 CRUD、赛事数据管理

## 技术栈

- 前端：Vite + React 18 + TypeScript + React Router v6
- 后端：Express + TypeScript + better-sqlite3
- 认证：JWT + bcryptjs

## 运行方式

```bash
cd server && npm install && cd ..
cd client && npm install && cd ..
npm run dev
```

前端启动在 http://localhost:5173，后端启动在 http://localhost:3001。

首次启动会自动创建 SQLite 数据库，插入教练和管理员账号以及示例训练模板。
